# QueueMaster - Система электронной очереди

## Описание
Программный комплекс для управления потоком посетителей. Включает в себя терминал выдачи талонов, информационное табло, рабочее место оператора и панель администратора.

## Технологии
* Python 3.10+
* Flask
* SQLite3
* Werkzeug
* HTML5 / CSS3 / JavaScript

## Установка и запуск
1. Скачать файл QueueMaster.zip
2. Разархивировать в пустую папку
3. Запустить файл START_SERVER
