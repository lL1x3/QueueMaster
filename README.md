# 🏛️ QueueMaster - Система электронной очереди

## Описание
Программный комплекс для управления потоком посетителей. Включает в себя терминал выдачи талонов, информационное табло, рабочее место оператора и панель администратора.

## Технологии
* Python 3.10+
* Flask (Web Framework)
* SQLite3 (База данных)
* Werkzeug (Хеширование паролей)
* HTML5 / CSS3 / JavaScript (Vanilla)

## Установка и запуск
1. Клонируйте репозиторий.
2. Установите зависимости: `pip install flask werkzeug`
3. Запустите сервер: `python app.py`
4. Откройте в браузере: `http://localhost:5000`
